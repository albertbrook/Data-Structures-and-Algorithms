### 链表操作
```
p               打印
pp              倒序打印
c               清空链表
```
### 指针
```
mp              移动到上一个结点
mp i            往上移动i个结点
mn              移动到下一个结点
mn i            往下移动i个结点
mm i            移动到索引为i的结点
```
### 增
```
i               进入insert模式
    q           退出insert模式
    f d         链表首结点插入d
    b d         链表尾结点插入d
    i i d       在链表索引为i的结点插入d
```
### 删
```
d               进入delete模式
    q           退出delete模式
    f           删除链表首结点
    b           删除链表尾结点
    d i         删除链表索引为i的结点
```
### 改
```
a d             修改当前指针所指结点的值
aa i d          修改索引为i的结点的值
```
### 查
```
sf              查询链表首结点的值
sb              查询链表尾结点的值
ss i            查询链表索引为i的结点的值
```
